F.3 IFC4 Addendum 2

The Second Addendum of the IFC4 release has the main scope to provide improvements to geometry issues that had been reported during the implementation of IFC4 and IFC4 Addendum 1. It resolves these issues and addresses further implementation issues and concerns.

The main changes incorporated into IFC4 ADD2 schema are:

\X\X\* new entities IfcPolygonalFaceSet, IfcIndexedPolygonalFace, IfcIndexedPolygonalFaceWithVoids, Ø\X\* adding IfcPolygonalFaceSet and IfcTriangulatedFaceSet to IfcBooleanOperand with the rule that only closed tessellations can be used, \X\X\* Improvement of advanced boundary representation, particularly for using elementary surfaces. Ø\X\* new entities for intersection curves to enable curves defined in u,v parametric space IfcSurfaceCurve, IfcIntersectionCurve, IfcSeamCurve,
