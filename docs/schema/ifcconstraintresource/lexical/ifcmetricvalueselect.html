Semantic definitions at the type
Type definition

IfcMetricValueSelect is a select type that enables selection of the data type for the value component of an IfcMetric.

Types are used as follows:

\\\\\* IfcValue: A constant value using project default units. \\\\\* IfcMeasureWithUnit: A constant value using specified units. \\\\\* IfcAppliedValue: A value calculated from a formula. \\\\\* IfcTable: A value retrieved from a table. \\\\\* IfcTimeSeries: A value that varies over time. \\\\\* IfcReference: A value referenced on an object attribute.
